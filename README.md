# Twitter Sentiment Analysis

This project performs sentiment analysis on Twitter data using Machine Learning and Natural Language Processing (NLP).

The model predicts whether a tweet is positive or negative using Logistic Regression and TF-IDF Vectorization.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK
* TF-IDF Vectorizer
* Logistic Regression

---

## Dataset

* Sentiment140 Dataset
* Dataset contains 1.6 million tweets.

---

## Features

* Text preprocessing
* Stopword removal
* Stemming
* TF-IDF Vectorization
* Sentiment classification
* Machine learning model training

---

## Machine Learning Model

* Logistic Regression

---

## Accuracy

* Achieved good accuracy on test data.

---

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Text Cleaning
4. Stemming
5. Train-Test Split
6. TF-IDF Vectorization
7. Model Training
8. Prediction and Evaluation
